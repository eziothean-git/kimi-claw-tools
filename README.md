# Kimi Claw Tools

Kimi Claw's automation tools for RL locomotion research tracking.

## 工具列表

### 1. Paper Intelligence Analyzer (`paper_analyzer.py`)
- 自动分析ArXiv论文的核心贡献
- 评估与你的研究方向的匹配度
- 提取关键技术、方法、实验结果

### 2. Daily Report Generator
- 整合GitHub动态、论文分析、趋势追踪
- 生成结构化的每日研究简报
